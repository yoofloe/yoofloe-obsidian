# Yoofloe MCP Wrapper

The Yoofloe MCP wrapper is an internal preview surface and is not part of the public Yoofloe Obsidian plugin launch.

Public users should install the Obsidian plugin from the Community Plugin Store and use `Settings -> Yoofloe` to connect an Obsidian `pat_yfl_...` token plus their own Google provider setup.

The release workflow must not publish a wrapper zip or public MCP setup guide until the CLI/MCP public launch gate is opened.
