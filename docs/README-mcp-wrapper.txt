Yoofloe Obsidian MCP Wrapper

This zip contains:
- mcp-server.js
- README-mcp-wrapper.txt

Requirements:
- Node 20+
- YOOFLOE_PAT=pat_yfl_...
- YOOFLOE_VAULT_PATH=/absolute/path/to/your/Obsidian vault

Quick start:
1. Unzip this package anywhere on your machine.
2. Set YOOFLOE_PAT and YOOFLOE_VAULT_PATH in your MCP client or shell.
3. Point your MCP client at the extracted mcp-server.js file.

Example:
node /absolute/path/to/mcp-server.js

Public install and troubleshooting hub:
https://www.yoofloe.com/docs/external-tools
