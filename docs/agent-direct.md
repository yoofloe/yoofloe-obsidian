# Yoofloe Agent Direct

Agent Direct is an internal preview surface and is not part of the public Yoofloe Obsidian plugin launch.

Public users should install the Obsidian plugin from the Community Plugin Store and use `Settings -> Yoofloe` to connect an Obsidian `pat_yfl_...` token plus their own Google provider setup.

Do not publish Agent Direct setup steps, MCP configuration snippets, or download links until the CLI/MCP public launch gate is opened.
